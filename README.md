# Quiz-react-app

<h4>Technology stack</h4>
<ul>
<li>React Js</li>
<li>Mysql</li>
<li>Lumen</li>
</ul>

<h4>Commands to install app</h4>
<ol>
<li>npm install</li>
<li>npm start</li>
</ol>

<h4>Commands to run backend</h4>
<ol>
<li>php -S localhost:8000 -t public</li>
</ol>
